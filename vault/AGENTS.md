# AGENTS

Este cofre é uma **wiki CCD** mantida por LLM, no padrão llm-wiki.

**Leia [`CLAUDE.md`](CLAUDE.md)** para todas as convenções: as três camadas (`raw/` imutável → `wiki/` gerada → schema), estrutura de pastas, tipos de nota e frontmatter, operações Ingest/Query/Lint, regras de linking (graph-first), e busca.

O conteúdo de `CLAUDE.md` é agente-agnóstico — vale para qualquer LLM (Claude Code, Codex, Gemini, etc.).
